# FP-FLOW [![Build Status](https://travis-ci.com/seb-bizeul/fp-flow.svg?branch=master)](https://travis-ci.com/seb-bizeul/fp-flow) 

## Install
    npm i -S @sbizeul/fp-flow

or

    yarn add @sbizeul/fp-flow

## Build
    npm run build

or

    yarn build

## Test
    npm run test

or

    yarn test
